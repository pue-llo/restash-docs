---
title: Install Restash on macOS
description: Build and install Restash from source on Apple Silicon Macs.
---

Restash is distributed as source. You build the `.app` once, then install
the bundle. Building gives you the real signed bundle, so every macOS
permission prompt says **"Restash"** instead of **"Electron"**.

**Requirements**

- macOS 12 (Monterey) or later
- Apple Silicon (M-series) Mac
- Node.js 18 or later (`node --version` to check)
- No Xcode required — the native helpers are prebuilt and committed.

**Build + install**

```bash
git clone https://github.com/<your-distribution-link>.git restash
cd restash
npm install
npm run dist

rm -rf /Applications/Restash.app
cp -R dist/mac-arm64/Restash.app /Applications/Restash.app
xattr -cr /Applications/Restash.app
open /Applications/Restash.app
```

The build also writes a drag-to-Applications DMG at
`dist/Restash-1.1.0-arm64.dmg` if you'd rather install that way.

Restash now lives in the **menu bar** and the **Dock**. Press **⌃⇧V** to
open the numpad at your cursor.

**Trouble?** See [App won't open / "damaged" warning](/troubleshooting/app-wont-open/) below.
